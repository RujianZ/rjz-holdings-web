  - "Most tools record tasks you've already extracted, or chat at you. Life Admin OS reads the source material itself and returns a structured action card — what it is, how urgent, the deadline, key facts and risks, the suggested next step, and a copy-pasteable reply if one is called for."
  - "All AI runs against a local LM Studio instance (Gemma 3n) on the user's machine. Data lives in IndexedDB. Nothing leaves the device."
